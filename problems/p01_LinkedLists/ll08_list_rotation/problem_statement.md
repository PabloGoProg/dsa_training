## 08: Rotar una Lista Enlazada
Dada una lista enlazada (list), rota la lista a la derecha en k posiciones, donde k es un nÃºmero entero no negativo.

**Ejemplos**

```python
# Ejemplo 1
Entradas: head = [1, 2, 3, 4, 5], k = 2
Salida esperada: [4, 5, 1, 2, 3]

# Ejemplo 2
Entradas: head = [0, 1, 2], k = 4
Salida esperada: [2, 0, 1]

# Ejemplo 3
Entradas: head = [1, 2], k = 0
Salida esperada: [1, 2]

# Ejemplo 4
Entradas: head = [1, 2, 3], k = 3
Salida esperada: [1, 2, 3]

# Ejemplo 5
Entradas: head = [], k = 1
Salida esperada: []
```
